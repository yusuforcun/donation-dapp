## 🎯 Overview

This DApp demonstrates how to build a donation system on Ethereum where users can:
- Connect their MetaMask wallet
- Make donations in ETH
- View real-time donation statistics
- Track contract balance

All transactions are recorded on the blockchain, ensuring transparency and immutability.

## ✨ Features

- 🔗 **MetaMask Integration** - Seamless wallet connection
- 💝 **Instant Donations** - Send ETH directly to smart contract
- 📊 **Real-time Stats** - Live updates of:
  - Total donations received
  - Number of unique donors
  - Contract balance
- ✅ **Transaction Status** - Real-time feedback on donation status
- 🎨 **Responsive Design** - Works on desktop and mobile devices
- 🔒 **Secure** - All transactions are blockchain-verified
